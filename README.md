##  Project Overview

The **Urban Planning Research Tool** is an interactive web application built using **Streamlit** that analyzes and visualizes city-level urban datasets.  
It helps understand population trends, zoning patterns, and transportation data to support **urban planning research and data-driven decision making**.

This project demonstrates a complete workflow from **data handling to live deployment**.

---

##  Objectives of the Project

- Analyze **urban population data** using census datasets  
- Study **zoning and land-use patterns**  
- Explore **transportation statistics**  
- Provide an **interactive dashboard** for urban data exploration  
- Demonstrate end-to-end project development:
  - Dataset selection
  - Data processing
  - Visualization
  - Deployment

---

##  Datasets Used

The datasets are stored in the `data/` folder:

| Dataset | Description |
|------|-----------|
| `census.csv` | Population and demographic statistics |
| `zoning.csv` | Land use and zoning information |
| `transport.csv` | Transportation and mobility data |
